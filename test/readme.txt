TO START, OPEN newserver.py IN A NEW TERMINAL WINDOW

NEXT, OPEN AN INSTANCE OF newclient.py IN ANOTHER TERMINAL WINDOW
    ENTER "1" WHEN PROMPTED FOR PLAYER NUMBER

NEXT, OPEN ANOTHER INSTANCE OF newclient.py IN ANOTHER TERMINAL WINDOW
(THERE SHOULD BE 3 WINDOWS NOW--1 SERVER, AND 2 CLIENTS)
    ENTER "2" WHEN PROMPTED FOR PLAYER NUMBER

CHOOSE CLIENT WINDOW OF YOUR CHOICE, AND SEND A MESSAGE

ON THE OTHER CLIENT WINDOW, TYPE "RECV" TO GET THE TRANSMISSION FROM THE OTHER CLIENT

TO QUIT, ENTER "quit" IN BOTH THE CLIENTS

TO QUIT THE SERVER, USE CNTRL-C
